--- title: Top Nav navURL: http://google.com navMenuLabel: Menu styles: Test! notes: | ![topnav](./assets/strapless/images/ui_components/navigation/topnav.png) ### Top Nav #### *Primary Navigation* ### Usage Used for moving the user across top-level website pages, or sections within a single-page site (like pa.gov) To ensure navigation is always available to the user, the topnav remains fixed to the top of the page, on mobile and desktop. ![Topnav Fixed](./assets/strapless/images/layout/navPatterns_topnav_fixed.png) ![Scrolling](./assets/strapless/images/layout/scrolling.gif) Navigation is reserved for moving a user from one point in the site to another. It should remain consistent across every page of a given site, and should adapt to the screen size to ensure navigation is equally possible for every user. ### Appearance Navigation should be visually distinct and visually constant. Meaning, if the navigation is at the top on the home page, it should be there on a subpage. As with buttons, provided visual distinction and consistency are observed, there is no need to underline the text. It's acceptable to use buttons within the topnav, like the "Escape" button in the example. It should fill the entire height and width of a nav-item. Navigation items can also have assocated icons, though it's not required. In general, you should not put any external links in the navigation.
