{{#markdown}} ## Vertical Navigation The Web App Standards's design system, along with most modern websites, is built for vertical navigation. Meaning, they're designed to guide the user's focus _down the page_, while ensuring navigation is always immediately apparent and available. - user's both expect and prefer to scroll on a website (versus zooming in) - the visual positioning, weight and vertical scale are all intended to subtly encourange the user to scroll. {{/markdown}}
