Welcome to the Commonwealth's Web App Standards UI Library

What is this?

A set of consistent, reusable components developed as a collaboration between PAI, the Governor’s Communications Department, and various Commonwealth agencies. In addition to providing a common look and feel, these components are accessible, responsive, and scalable. Design standards are largely based on U.S. Web Design Standards (developed by 18F), and informed by currently accepted best practices.
