{{#markdown}} Navigation is used to move a user from one part of a website to another. It's essential that users are able to access the parts of a website they're seeking, in the fewest steps possible, from any other part of the site. It breaks down into four categories: - **Primary Navigation**: used to bring users to each top level parent page - Top Nav - **Secondary Navigation** - Side Nav - Sidebar Nav - **Tertiary Navigation** - Pagination - Breadcrumbs - Buttons - Hyperlinks - **Search** - Search Input - Search Results {{/markdown}}
